## 安装Jekyll
```sh
sudo gem install jekyll
sudo gem install
```

## 博客常用命令
```sh
# 当前目录创建博客
jekyll new 博客名
jekyll new .

# 为了使博客不出问题我们先安装依赖
sudo bundle install
bundle exec jekyll serve
sudo bundle install
```
